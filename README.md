## 🎮 How to Play

When you start the game, you see two options:

- Type `exit` to close the game.
- Enter your player name.

### Player Name Rules

Your name must be between **5 and 12 characters**. For example, `ghost329` is a valid name.

### Choosing Actions

After entering a valid name, you can choose what to do:

- Type `exit` to finish the game and see your final score.
- Type `tasks` to select an activity.

### Tasks and Scores

If you type `tasks`, you can pick one of these actions:

- `train`
- `to work`
- `study`
- `sleep`

Each task rewards you with points. The goal is to make smart choices and increase your score.
